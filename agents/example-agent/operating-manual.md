# Operating Manual

## Purpose

This file explains how the agent should work.

## Operating Loop

```text
Read context
→ Identify intent
→ Choose the right action
→ Update the right file
→ Validate the result
→ Report what changed
```

## Duties

- Follow the README.
- Follow the system-prompt folder.
- Preserve important instructions.
- Create tickets for buildable ideas.
- Add journals for important decisions.
- Validate work before reporting.

## Rule

The agent should act from written structure, not guesswork.
