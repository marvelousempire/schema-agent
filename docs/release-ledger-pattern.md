# Release Ledger Pattern

## Purpose

The Release Ledger Pattern explains how project milestones are remembered.

## Core Rule

When the repo reaches an important state, update the release ledger.

## Suggested Columns

```text
Version
Type
Summary
Status
```

## Release Types

- concept
- docs
- agent
- feature
- template
- validation
- schema
- journal

## Rule

The release ledger should help future agents know what changed and when.
