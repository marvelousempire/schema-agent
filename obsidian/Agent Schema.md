# Agent Schema

## Purpose

The Agent Schema is the reusable pattern for creating agents.

## Core Idea

An agent is a living system, not only a prompt.

## Main Parts

- Intent
- Purpose
- Role
- Rules
- Firing order
- Knowledge sources
- Feature tickets
- Templates
- Journals
- Validation
- Release memory
