# FEAT-0006 - New Agent Checklist

## Status

Draft

## Valid Intent

Give builders a clear checklist for creating a new agent repo.

## User Value

A new agent can be created without forgetting important files, folders, or validation steps.

## Description

Create a checklist that covers root files, agent folder files, prompt stack files, ledgers, wiki notes, journals, and validation requirements.

## Acceptance Criteria

- Checklist exists.
- Required root files are listed.
- Required agent files are listed.
- Required prompt files are listed.
- Validation criteria are listed.
