# FEAT-0002 - Firing Order Pattern

## Status

Draft

## Valid Intent

Give each agent a clear order for loading its instructions.

## User Value

The agent knows where to begin and does not guess its operating order.

## Description

Define the standard system-prompt folder and numbered firing-order files for agents.

## Acceptance Criteria

- Firing order guide exists.
- System prompt folder pattern is defined.
- Machine-readable firing order is recommended.
